# Security Policy

## Supported Versions

Currently, only the `main` branch (latest stable release) is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| Main    | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of NCET Buddy seriously.

**Please do not report security vulnerabilities through public GitHub issues.**

If you believe you have found a security vulnerability in NCET Buddy, please report it via email to the core maintainers/organization privately. Include a detailed description of the vulnerability and steps to reproduce. 

We will acknowledge your report within 48 hours and we will do our best to provide a timeline for a fix. We ask that you do not publish details of the vulnerability until we have had time to address it and deploy a patch.
